# tree-sitter-10x

Tree-sitter grammar scaffold for the 10x language.

## Scope

- directive-first syntax (`@signal`, `@render`, `@html`, `@on`, etc.)
- assignment/lambda blocks
- tags with attrs/interpolation
- symbols, numbers, strings, and collections

## Commands

```sh
npm install
npm run generate
npm run test
```

## Queries

- `queries/highlights.scm`
- `queries/brackets.scm`
- `queries/outline.scm`
