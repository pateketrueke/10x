# Math Utilities Module

Exportable math functions.

## Exports

@export add, subtract, multiply, divide, square, cube.

## Implementation

add = (a b) -> a + b.
subtract = (a b) -> a - b.
multiply = (a b) -> a* b.
divide = (a b) -> a / b.
square = x -> x* x.
cube = x -> x* x* x.

## Constants

PI = 3.14159.
E = 2.71828.

@export PI, E.
