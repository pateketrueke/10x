10x
Markdown-native programming language
Toy language · Reactive runtime · Compiler

Write code that reads like notes.

10x is a markdown-native language that interprets expressions inline, compiles to ESM modules, and renders reactive web components from the same .md source file.

View on GitHub

Language

Numbers, units, ranges, symbols.

1/2.
5 cm to :in.
[1..5].
:ok.

Functions

Lambdas, closures, and pipe composition.

add = (a b) -> a + b.
double = n -> n * 2.
[1..5] |> map(double).

Live Editor

Compile/eval in the browser.

Signals & Reactivity

Reactive state, derived values, and automatic re-render.

count = @signal 0.
double = @computed count * 2.
@render "#app" @html
  [<h1>{count}</h1>, <p>{double}</p>].
@on "click" "#inc"
  count = count + 1.

Web Components

Shadow DOM components compile to setup(host).

count = @signal @prop "start" 0.
@render @shadow @html
  <div><h1>{count}</h1></div>.
@on "click", "#inc",
  count = count + 1.

Compiler

Compile .md to portable ESM.

# source                    # compiled
count = @signal 0.      -> export const count = Runtime.signal(0);
@render "#app" @html    -> Runtime.render("#app", Runtime.html(() => ...));
@on "click" "#btn"      -> Runtime.on("click", "#btn", () => ...);

CSS

Raw CSS, object styles, and auto atomic utilities.

@style "body { margin: 0; }".
@style (:host { backgroundColor: "#0f1217" }).
@render "#app" @html
  <div class="flex items-center gap-2 p-4">x</div>.

VDOM Directives

Native somedom attrs: d:*, s:*, and ref.

open = @signal :on.
name = @signal "".
el = @signal :nil.
@render "#app" @html
  <input d:model={name} d:show={open} ref={el} />.

Bundler

Bundle local modules and aliases into a single app artifact.

10x compile app/main.md --bundle --out app.js
10x compile app/main.md --bundle --alias "@app=./src"
bun run app.js
node app.js

Modular Runtime

Import only what you need.

10x/runtime/core

  • signal, effect, read
  • computed, batch, untracked
  • Pure JS, no DOM

10x/runtime/dom

  • render, renderShadow
  • on, style, h
  • Browser-only layer

10x/prelude

  • head, tail, map, filter
  • take, range, concat
  • Works in JS runtimes

Execution Pipeline

.md source scanner parser eval result compiler .js ESM
  • Markdown prose is ignored by eval and emitted as comments by compile.
  • Interpreter and compiler share parser semantics.
  • Runtime split: pure core + browser DOM layer.
