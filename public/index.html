10x
Markdown-native programming language
Language + Runtime

Write code that reads like notes.

10x is a toy language in JavaScript with units, ranges, fractions, symbols, and a markdown-first source format. Build expressions inline, keep prose around them, and execute from CLI, REPL, or browser demo.

Execution Pipeline

source text scanner parser AST eval result
  • Markdown-aware syntax with `.md` source files.
  • Shared environment supports chained statements and imports.
  • Built-in unit and currency conversion support.

Live Demo

Core Ideas

  • Four primitives: string, number, symbol, range.
  • Units as first-class numeric wrappers with conversion methods.
  • Inline interpolation across strings and markup.
  • REPL and CLI with import/export across markdown modules.

Quick Start

make eval code="1 + 2"
